本章介绍半经典光学理论，其中电磁场用经典方式描述，而物质系统用量子力学处理。这种方法成功解释了大多数激光-原子相互作用现象，包括吸收、受激发射、拉比振荡和饱和效应。我们将建立从经典渲染到量子光学的桥梁，为理解现代光学现象提供必要的理论基础。

## 21.1 经典场与量子化物质基础

### 21.1.1 半经典近似的物理动机

在许多光学现象中，光场强度足够大，使得光子数不确定度相对较小。此时，可以将光场视为经典电磁波，而保持原子系统的量子描述。这种半经典近似在以下条件下有效：

1. **大光子数条件**：⟨n̂⟩ ≫ 1，其中 n̂ 是光子数算符
2. **相干态近似**：激光场接近相干态 |α⟩
3. **忽略量子涨落**：Δn/⟨n⟩ ≪ 1

### 21.1.2 经典电磁场描述

考虑单色平面波：

**E**(r, t) = **E**₀ cos(k·r - ωt + φ) = ½**E**₀[e^{i(k·r - ωt + φ)} + c.c.]

其中：
- **E**₀ 是电场振幅矢量
- k 是波矢
- ω 是角频率
- φ 是初相位

在偶极近似下（原子尺度 ≪ 波长），可以忽略空间变化：

**E**(t) = **E**₀ cos(ωt - φ)

### 21.1.3 原子的量子力学描述

考虑具有离散能级的原子系统。系统状态由波函数描述：

|ψ(t)⟩ = Σₙ cₙ(t)e^{-iEₙt/ℏ}|n⟩

其中 |n⟩ 是能量本征态，Eₙ 是对应能量。

密度算符形式：

ρ̂(t) = |ψ(t)⟩⟨ψ(t)|

密度矩阵元素：

ρₘₙ(t) = ⟨m|ρ̂(t)|n⟩ = cₘ(t)c*ₙ(t)e^{-i(Eₘ-Eₙ)t/ℏ}

### 21.1.4 相互作用哈密顿量

在电偶极近似下，光-物质相互作用哈密顿量为：

Ĥᵢₙₜ = -**d̂**·**E**(t)

其中 **d̂** = -e**r̂** 是电偶极矩算符。

总哈密顿量：

Ĥ = Ĥ₀ + Ĥᵢₙₜ = Ĥ₀ - **d̂**·**E**(t)

### 21.1.5 旋转波近似

在近共振条件下（|ω - ω₀| ≪ ω₀），可以采用旋转波近似（RWA），忽略快速振荡项。相互作用哈密顿量简化为：

Ĥᵢₙₜ^{RWA} = -ℏΩ/2(σ̂₊e^{-iωt} + σ̂₋e^{iωt})

其中：
- Ω = **d**·**E**₀/ℏ 是拉比频率
- σ̂₊, σ̂₋ 是升降算符

## 21.2 二能级系统与Bloch方程

### 21.2.1 二能级系统模型

考虑最简单的量子系统：二能级原子，基态 |g⟩ 和激发态 |e⟩：

- 能量：Eg = 0, Ee = ℏω₀
- 跃迁频率：ω₀ = (Ee - Eg)/ℏ
- 偶极矩：**d** = ⟨e|**d̂**|g⟩

### 21.2.2 密度矩阵演化

密度矩阵形式：

ρ̂ = (ρee  ρeg)
    (ρge  ρgg)

满足约束：
- Tr(ρ̂) = ρee + ρgg = 1（归一化）
- ρge = ρ*eg（厄米性）
- 0 ≤ ρee, ρgg ≤ 1（正定性）

### 21.2.3 光学Bloch方程推导

从von Neumann方程出发：

iℏ∂ρ̂/∂t = [Ĥ, ρ̂]

在旋转坐标系中，定义：

ρ̃eg = ρeg e^{iωt}
ρ̃ge = ρge e^{-iωt}

得到光学Bloch方程：

dρee/dt = iΩ/2(ρ̃eg - ρ̃ge) - Γρee
dρ̃eg/dt = iΔρ̃eg + iΩ/2(ρee - ρgg) - γρ̃eg

其中：
- Δ = ω - ω₀ 是失谐
- Γ = 1/T₁ 是纵向弛豫率
- γ = 1/T₂ 是横向弛豫率

### 21.2.4 Bloch矢量表示

定义Bloch矢量分量：

u = ρ̃eg + ρ̃ge = 2Re(ρ̃eg)
v = i(ρ̃eg - ρ̃ge) = -2Im(ρ̃eg)
w = ρee - ρgg

Bloch方程的矢量形式：

du/dt = Δv - u/T₂
dv/dt = -Δu + Ωw - v/T₂
dw/dt = -Ωv - (w - w₀)/T₁

其中 w₀ = -1 是热平衡时的布居数差。

### 21.2.5 Bloch球几何

Bloch矢量 **R** = (u, v, w) 在单位球内运动：

|**R**|² = u² + v² + w² ≤ 1

- 纯态：|**R**| = 1（球面上）
- 混合态：|**R**| < 1（球内部）
- 最大混合态：**R** = 0（球心）

## 21.3 拉比振荡与光学章动

### 21.3.1 共振情况（Δ = 0）

在精确共振时，Bloch方程简化为：

du/dt = -u/T₂
dv/dt = Ωw - v/T₂
dw/dt = -Ωv - (w - w₀)/T₁

忽略弛豫（T₁, T₂ → ∞），得到：

d²w/dt² + Ω²w = 0

解为：

w(t) = cos(Ωt)
v(t) = -sin(Ωt)
u(t) = 0

这描述了布居数的拉比振荡。

### 21.3.2 失谐情况（Δ ≠ 0）

考虑失谐但无弛豫的情况，定义广义拉比频率：

Ω' = √(Ω² + Δ²)

布居数演化：

w(t) = (Δ²/Ω'²) + (Ω²/Ω'²)cos(Ω't)

最大激发概率：

Pₘₐₓ = Ω²/(Ω² + Δ²)

### 21.3.3 脉冲面积定理

对于时变电场 E(t)，定义脉冲面积：

θ = ∫_{-∞}^{∞} Ω(t)dt = (1/ℏ)∫_{-∞}^{∞} **d**·**E**(t)dt

特殊脉冲：
- π脉冲：θ = π，完全反转布居数
- 2π脉冲：θ = 2π，返回初态
- π/2脉冲：θ = π/2，创建相干叠加态

### 21.3.4 章动现象

在旋转坐标系中，Bloch矢量绕有效磁场 **Ω**ₑff = (Ω, 0, Δ) 进动：

d**R**/dt = **Ω**ₑff × **R** - **Γ**·**R**

其中 **Γ** 是弛豫张量。

### 21.3.5 绝热跟随

当场参数缓慢变化时（|dΩ/dt| ≪ Ω²），系统绝热跟随瞬时本征态：

w_ad(t) ≈ -Δ(t)/Ω'(t)

这是绝热快速通道（ARP）和STIRAP技术的基础。

## 21.4 饱和与功率展宽

### 21.4.1 稳态解

在连续波激发下，设 d/dt = 0，得到稳态解：

u₀ = -2ΩΔT₂²/(1 + Δ²T₂² + Ω²T₁T₂)
v₀ = -2ΩT₂/(1 + Δ²T₂² + Ω²T₁T₂)
w₀ = -(1 + Δ²T₂²)/(1 + Δ²T₂² + Ω²T₁T₂)

### 21.4.2 饱和强度

定义饱和参数：

s = Ω²T₁T₂/(1 + Δ²T₂²) = I/Iₛₐₜ

其中饱和强度：

Iₛₐₜ = ℏ²/(2|**d**|²T₁T₂) × (1 + Δ²T₂²)

稳态激发态布居数：

ρee = s/(2(1 + s))

### 21.4.3 功率展宽

吸收线型：

L(ω) = (γ'/2π)/[(ω - ω₀)² + (γ'/2)²]

其中展宽的线宽：

γ' = γ√(1 + s) = γ√(1 + I/Iₛₐₜ)

这就是功率展宽效应。

### 21.4.4 均匀与非均匀展宽

**均匀展宽**：所有原子具有相同跃迁频率
- 自然展宽：γₙ = Γ/2
- 碰撞展宽：γc ∝ p/√T

**非均匀展宽**：原子跃迁频率分布
- 多普勒展宽：Δωᴅ = ω₀√(2kT/mc²)
- 晶格无序：场的空间不均匀性

### 21.4.5 烧孔效应

在非均匀展宽介质中，强激光选择性激发特定速度类原子，在吸收谱中"烧"出一个孔：

孔宽度：Δωₕₒₗₑ ≈ γ'
孔深度：∝ s/(1 + s)

## 21.5 从经典到量子的过渡

### 21.5.1 半经典理论的局限性

半经典理论无法解释：
1. **自发辐射**：需要量子化光场
2. **光子统计**：亚泊松分布、压缩态
3. **量子关联**：纠缠、非经典关联
4. **真空涨落**：Casimir效应、Lamb位移

### 21.5.2 自发辐射的唯象引入

在半经典框架中，通过唯象地引入自发辐射率 A₂₁：

dρee/dt|ₛₚ = -A₂₁ρee

Einstein A系数与偶极矩的关系：

A₂₁ = ω₀³|**d**|²/(3πε₀ℏc³)

修正的Bloch方程：

dw/dt = -Ωv - (w + 1)/T₁ - A₂₁(w + 1)/2

### 21.5.3 速率方程近似

当失谐 |Δ| ≫ γ 或强场 Ω ≫ γ 时，相干项快速衰减，可采用速率方程：

dρee/dt = R₁₂ρgg - R₂₁ρee - A₂₁ρee
dρgg/dt = -R₁₂ρgg + R₂₁ρee + A₂₁ρee

其中受激跃迁率：

R₁₂ = R₂₁ = (Ω²/4)·(γ/2π)/[(ω - ω₀)² + (γ/2)²]

### 21.5.4 与体积渲染方程的联系

将原子介质视为具有频率依赖吸收/发射的体积：

dL(ω)/ds = -σₐ(ω)n[1 - ρee(ω)]L(ω) + σₑ(ω)nρee(ω)Lₑ(ω)

其中：
- σₐ(ω), σₑ(ω) 是吸收/发射截面
- n 是原子数密度
- ρee(ω) 是频率ω处的激发态布居

这建立了微观原子物理与宏观渲染方程的联系。

### 21.5.5 量子修正与展望

完整量子理论预测的修正：

1. **真空Rabi分裂**：强耦合腔QED
2. **共振荧光谱**：Mollow三峰
3. **光子反聚束**：g⁽²⁾(0) < 1
4. **纠缠光子对**：参量下转换

这些效应需要第27-28章的完整量子光学处理。

## 常见陷阱与错误

1. **旋转波近似的误用**
   - 错误：在强场或大失谐时仍使用RWA
   - 正确：检查条件 Ω, |Δ| ≪ ω₀

2. **忽略逆过程**
   - 错误：只考虑吸收，忽略受激发射
   - 正确：完整的Bloch方程自动包含两个过程

3. **弛豫时间的混淆**
   - 错误：认为T₁ = T₂
   - 正确：T₂ ≤ 2T₁（纯失相导致T₂ < 2T₁）

4. **稳态近似的不当使用**
   - 错误：对脉冲激发使用稳态解
   - 正确：检查脉冲宽度 ≫ T₁, T₂

5. **绝热条件的违反**
   - 错误：快速改变参数时假设绝热跟随
   - 正确：验证 |ω̇/ω²| ≪ 1
